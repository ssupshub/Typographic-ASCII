Method

— How a work is made

Every piece navigates the same four stages — from source image to final glyph arrangement. The machine handles measurement; the artist handles meaning.

01

Source & Luminance Mapping

Every work begins with an image reduced to a single-channel luminance field. This field is sampled at the resolution of the output character grid.

Tools: Custom Python pipeline · OpenCV · Manual study
02

Glyph Selection by Brightness & Width

Characters are chosen across two axes: brightness value and measured pixel width. Dense glyphs anchor dark regions; hairline glyphs breathe in the light.

No monospace assumptions — every character measured via canvas measureText()
03

Proportional Line Packing

Using proportional font measurement I pack glyphs tightly across each row, filling a target pixel width with characters whose widths sum exactly to that measure.

Implemented in TypeScript · Sub-pixel accuracy
04

Hand Refinement

The algorithm gives me a draft. Then I read it — character by character — adjusting awkward transitions and reinforcing edges that have become soft.

Refinement ratio: typically 8–15% of characters adjusted in final pass
