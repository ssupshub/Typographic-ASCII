— The Artist

Text
Made
Visible

I work in the space between language and image — where a glyph is not merely a letter but a unit of density, weight, and luminous form.

Using proportional typography instead of monospace grids, I measure the exact pixel width of each character to sculpt images from pure text. No pixels — only the alphabet, precisely placed.

"Every character earns its exact position. The image emerges from constraint, not freedom."
— Studio statement, 2022

The
Method

— How a work is made

Every piece navigates four stages — from source luminance to final glyph arrangement. The machine measures; the artist decides.

See the Results
01 · Eye / Iris
02 · Cascade flow
03 · Spiral vortex
01

Source & Luminance Mapping

Every work begins with an image reduced to a single-channel luminance field — the map I'll navigate with type. Shadows, gradients, the precise moment light catches an edge.

Sampled at character-grid resolution: 40–80 cols for print, 60–120 for screen.

Tools: Custom Python pipeline · OpenCV · Manual study
02

Glyph Selection by Brightness & Width

Characters chosen across two axes: brightness value and measured pixel width. Dense glyphs (M, #, @) anchor dark regions. Hairline glyphs (·, l, |) breathe in light.

Every character individually measured via canvas measureText()
03

Proportional Line Packing

Glyphs packed across each row filling a target pixel width precisely. Variable weight and italic variants assigned per-character — a bold M and a light i at the same point size render very differently.

TypeScript · Sub-pixel accuracy · Handles bidi, emoji, CJK
04

Hand Refinement

The algorithm produces a draft. Then I read it — character by character — adjusting transitions, replacing glyphs that read as noise rather than form. The final work is always partly hand-edited.

Refinement ratio: 8–15% of characters adjusted in final pass
